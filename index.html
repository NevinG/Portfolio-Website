Fake quizlet

I frequently found myself wishing quizlet had a different study plan for sets. I wanted to have control over when and how frequently certain cards were shown to me. So this lead me to make my own version of quizlet. It is made with React, Node JS, Express JS, and PostgreSQL. I have study sets locally on my PostgreSQL server that I use this web app to study with.

Practicetypingcode.com

Have you ever been disappointed that you type code much slower than your regular typing? Well this is a lightweight typing website where you can practice typing some short code statements in various languages. It isn't very robust, but a good way to kill time and get better at typing those tricky character so commonly used in coding.

Ishihara Plate Generator

I am colorblind and have always been fascinated by the color blind tests. One weekend I decided I'd make my own plate generator so I could mess around with the different colors and see how my colorblindness operates. I also have experimented with reverse colorblindness tests where it is easier for colorblind people to read the number than it is people with normal color vision.

Million March Madness Brackets

I don't really watch much basketball, but come March I'm all about beating my friends in a March Madness Bracket. I decided that if I had a million brackets that I would probably beat them, and who knows, maybe I would get one of the best brackets in the county. I used NodeJS to make a jsonfile that contained my million brackets. I used Express JS to host a server in my garage that handled CRUD requests to update the teams were still alive and to get the brackets. I then used HTML, CSS, and JS to make a website that displayed all the results of my brackets.
